# Preset Klasörü

Bu klasöre XMP preset dosyalarınızı ekleyin.

## Kullanım
1. `.xmp` dosyalarını bu klasöre kopyalayın
2. Uygulama otomatik olarak okuyacak ve preset listesine ekleyecek
3. Preset isimleri dosya adından alınır

## Örnek
- `Cinematic.xmp` → "Cinematic" preset'i
- `My Wedding Style.xmp` → "My Wedding Style" preset'i
